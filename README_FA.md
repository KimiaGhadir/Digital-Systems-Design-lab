# آزمایشگاه سیستم‌های دیجیتال

این مخزن شامل آزمایش‌های درس **طراحی سیستم‌های دیجیتال** است.

## آزمایش‌ها
- آزمایش ۱: طراحی مدارهای ترکیبی با استفاده از امکانات شماتیک
- آزمایش ۲: طراحی مدارهای ترتیبی با استفاده از امکانات شماتیک
- آزمایش ۳: توصیف جریان داده
- آزمایش ۴: توصیف رفتاری
- آزمایش ۵: طراحی ضرب‌کننده
- آزمایش ۶: به‌زودی
- آزمایش ۷: به‌زودی
- آزمایش ۸: به‌زودی
- آزمایش ۹: به‌زودی
- آزمایش ۱۰: به‌زودی

## ساختار مخزن
هر آزمایش در یک پوشه جداگانه قرار می‌گیرد.

## هدف
هدف این مخزن مستندسازی طراحی و پیاده‌سازی آزمایش‌های آزمایشگاه سیستم‌های دیجیتال است.
